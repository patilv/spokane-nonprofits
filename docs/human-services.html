Human Services Focus

The organizations that meet basic community needs — food, housing, employment, youth, safety

Key finding 768 active human services nonprofits (NTEE groups I, J, K, L, O, P) serve the region, generating about $734 million a year — led by Second Harvest Inland Northwest, Goodwill Industries, and Spokane United Methodist Homes. The mix skews heavily toward general human services and youth development; employment services (44 orgs) is the smallest category, and housing & shelter (75 orgs) remains thin relative to the scale of housing need — housing organizations are among the region’s smallest by revenue.

Formation trend: is capacity keeping up?

New human-services organizations formed per year across the region, among organizations still active today (IRS ruling dates; dissolved organizations are not counted, so early years understate true formation activity).

hsHist = {
  // formations.csv is aggregated across all categories, so recompute from the
  // active-org file for the human services subset (active orgs only).
  const rows = orgs.filter(d => hsLetters.has(d.ntee_major) && d.ruling_year >= 1970 && d.ruling_year <= 2025);
  const roll = d3.rollups(rows, v => v.length, d => d.ruling_year)
    .map(([year, n]) => ({year, n}))
    .sort((a, b) => a.year - b.year);
  for (let i = 0; i < roll.length; i++) {
    const w = roll.slice(Math.max(0, i - 4), i + 1);
    roll[i].ma5 = d3.mean(w, d => d.n);
  }
  return roll;
}

})

viewof hsCat = Inputs.select(
  ["All six categories"].concat(Array.from(hsLetters.values())),
  {label: "Category"}
)
viewof hsCounty = Inputs.select(
  ["All counties"].concat(Array.from(new Set(hsOrgs.map(d => d.county))).sort()),
  {label: "County"}
)
viewof hsSearch = Inputs.search(
  hsOrgs.filter(d =>
    (hsCat === "All six categories" || hsLetters.get(d.ntee_major) === hsCat) &&
    (hsCounty === "All counties" || d.county === hsCounty)
  ),
  {placeholder: "Search names, cities…", columns: ["name", "city"]}
)
Inputs.table(
  hsSearch.map(d => ({
    Name: d.name,
    City: d.city,
    County: d.county,
    Category: hsLetters.get(d.ntee_major),
    NTEE: d.ntee ?? "—",
    "Reported revenue": d.revenue > 0 ? fmoney(d.revenue) : "—"
  })),
  {rows: 15, layout: "auto"}
)
